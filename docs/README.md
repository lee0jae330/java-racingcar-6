# 기능 구현

## 사용자 입력

- [ ] 경주할 자동차 이름 입력 받기
- [ ] 입력 받은 자동차 이름들 쉼표(,)로 구분하기
- [ ] 시도 횟수 입력 받기

## 자동차 경주

- [ ] 입력 받은 각 자동차마다 0~9 난수 생성하기
- [ ] 해당 난수가 전진 조건(4 이상)을 만족하면 전진하기
- [ ] 1,2번 과정이 끝나면 각 자동차들의 이동 상태 출력
- [ ] 위 과정을 입력 받은 시도 횟수만큼 반복

## 경주 종료

- [ ] 경주 우승자 출력

## 입력 조건

- [ ] 자동차 이름은 5자 이하
- [ ] 사용자가 잘못된 입력을 했을 시 'IllegalArgumentExcpetion' 발생 후 프로그램 종료
- [ ] 자동차 이름, 시도 횟수는 공백 X
